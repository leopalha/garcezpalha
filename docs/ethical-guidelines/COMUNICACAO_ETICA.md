# Guidelines de Comunicacao Etica - Garcez Palha

**Data:** 2025-12-23
**Versao:** 1.0
**Objetivo:** Garantir comunicacao etica em todos os canais

---

## PRINCIPIOS FUNDAMENTAIS

### 1. Honestidade Radical

> "Prefira perder um cliente a criar uma expectativa falsa."

- Diga a verdade sobre prazos
- Explique riscos e incertezas
- Nao oculte complexidades
- Admita quando nao souber

### 2. Transparencia Total

> "O cliente deve saber exatamente o que esta contratando."

- Precos claros desde o inicio
- Etapas do processo explicadas
- Riscos e probabilidades informados
- Custas e despesas destacadas

### 3. Respeito ao Cliente

> "Trate o cliente como gostaria de ser tratado em situacao vulneravel."

- Sem pressao excessiva
- Tempo para decidir
- Linguagem acessivel
- Empatia com a situacao

### 4. Profissionalismo

> "Somos advogados, nao vendedores de promessas."

- Qualidade sobre quantidade
- Etica sobre lucro
- Reputacao sobre conversao
- Longo prazo sobre imediato

## COMUNICACAO POR CANAL

### Website / Landing Pages

**TOM:** Institucional, confiavel, profissional

**FAZER:**
- Destacar tradicao e experiencia
- Mostrar credenciais verificaveis
- Explicar processo de trabalho
- Oferecer analise gratuita

**NAO FAZER:**
- Promessas de resultado
- Contagem regressiva falsa
- "Ultimas vagas" ficticio
- Pressao por escassez

**EXEMPLO BOM:**
```
"364 Anos de Tradicao em Solucoes Juridicas
Tecnologia de ponta aliada a experiencia de quem
ja resolveu milhares de casos."
```

**EXEMPLO RUIM:**
```
"ULTIMAS 3 VAGAS! Resolva seu problema em 72h!
Garantia de resultado ou dinheiro de volta!"
```

---

### Chat IA / Qualificacao

**TOM:** Acolhedor, esclarecedor, objetivo

**FAZER:**
- Identificar que e atendimento automatizado
- Coletar informacoes relevantes
- Encaminhar para especialista quando necessario
- Explicar proximos passos

**NAO FAZER:**
- Fingir ser advogado humano
- Dar parecer juridico definitivo
- Prometer resultados
- Pressionar por contratacao

**EXEMPLO BOM:**
```
"Ola! Sou a assistente virtual do escritorio Garcez Palha.
Vou fazer algumas perguntas para entender seu caso e
encaminhar para o especialista adequado.
O que aconteceu?"
```

**EXEMPLO RUIM:**
```
"Ola! Vou resolver seu problema agora!
Me conta o que houve que ja te garanto a vitoria!"
```

---

### WhatsApp

**TOM:** Proximo, rapido, mas profissional

**FAZER:**
- Responder rapidamente
- Ser claro e direto
- Confirmar entendimento
- Agendar contato formal

**NAO FAZER:**
- Dar consultoria pelo chat
- Prometer sem analise
- Usar emojis excessivos
- Enviar audios longos

**EXEMPLO BOM:**
```
"Ola, [Nome]! Recebemos sua mensagem.
Para analisar seu caso adequadamente,
nosso especialista em [area] entrara
em contato em ate 24h.
Posso confirmar o melhor horario?"
```

**EXEMPLO RUIM:**
```
"Ola!! Relaxa que vamos resolver seu problema rapidinho!!
So precisa contratar! Link de pagamento aqui!"
```

---

### Email

**TOM:** Formal, completo, documentado

**FAZER:**
- Assunto claro e objetivo
- Informacoes completas
- Proximos passos definidos
- Assinatura profissional

**NAO FAZER:**
- Subject lines clickbait
- Urgencia artificial
- Anexos suspeitos
- Falta de identificacao

**EXEMPLO BOM:**
```
Assunto: Proposta Comercial - Caso de Usucapiao | Garcez Palha

Prezado(a) [Nome],

Conforme conversamos, segue proposta para atuacao
em seu caso de usucapiao...

[Detalhes completos]

Atenciosamente,
Dr. [Nome]
OAB/RJ XXXXX
```

**EXEMPLO RUIM:**
```
Assunto: URGENTE!!! NAO PERCA SUA CHANCE!!!

Oi!! Precisa decidir AGORA ou vai perder tudo!!
Clica no link e paga!!
```

### Propostas Comerciais

**TOM:** Tecnico, transparente, completo

**ESTRUTURA OBRIGATORIA:**

1. **Identificacao das Partes**
   - Dados do cliente
   - Dados do escritorio

2. **Descricao do Caso**
   - Fatos apresentados
   - Documentos analisados

3. **Estrategia Proposta**
   - Medidas a serem adotadas
   - Ordem de acoes

4. **Prazos Realistas**
   - Prazo de atuacao (controlavel)
   - Prazo estimado do judiciario (incontrolavel)
   - Disclaimer obrigatorio

5. **Honorarios**
   - Valor total
   - Forma de pagamento
   - Custas e despesas a parte
   - O que esta incluido
   - O que NAO esta incluido

6. **Riscos e Probabilidades**
   - Analise honesta das chances
   - Possiveis obstaculos
   - Cenarios alternativos

7. **Disclaimers**
   - Dependencia de decisao judicial
   - Impossibilidade de garantia de resultado

---

### Follow-up

**TOM:** Prestativo, nao insistente

**REGRAS:**
- Maximo 3 follow-ups
- Espacamento adequado (3, 7, 15 dias)
- Sempre oferecer valor, nao so cobrar decisao
- Respeitar silencio como resposta

**EXEMPLO BOM:**
```
"Ola [Nome], tudo bem?

Notei que ainda nao tivemos retorno sobre a proposta.
Ficou alguma duvida que eu possa esclarecer?

Se preferir, podemos agendar uma ligacao rapida.
Caso nao tenha mais interesse, sem problemas -
ficamos a disposicao para o futuro.

Abracos,
[Assinatura]"
```

**EXEMPLO RUIM:**
```
"Ola!! Nao esqueceu de nos ne??
Precisa decidir HOJE senao nao vai dar mais!!
Vou ligar toda hora ate voce atender!!"
```

## TRATAMENTO DE OBJECOES

### "Esta muito caro"

**RESPOSTA ETICA:**
```
"Entendo sua preocupacao com o investimento.
Nossos honorarios refletem a complexidade do caso
e a experiencia de nossa equipe.

Posso explicar o que esta incluido e mostrar
opcoes de parcelamento?"
```

**NAO DIZER:**
```
"Se nao pagar vai perder tudo!"
"Outros advogados vao te cobrar o dobro!"
"E isso ou nada!"
```

### "Preciso pensar"

**RESPOSTA ETICA:**
```
"Claro, e uma decisao importante.
Fico a disposicao para qualquer duvida.
A proposta e valida por 15 dias."
```

**NAO DIZER:**
```
"Pensar em que? Vai perder a oportunidade!"
"So consigo esse preco ate hoje!"
"Se nao decidir agora outro cliente vai pegar sua vaga!"
```

### "Outro advogado prometeu resultado"

**RESPOSTA ETICA:**
```
"Entendo. Infelizmente, nenhum advogado pode
garantir resultado de processo judicial -
isso viola o Codigo de Etica da OAB.

O que podemos garantir e qualidade na atuacao,
transparencia no processo e dedicacao ao seu caso."
```

**NAO DIZER:**
```
"Eles estao mentindo!"
"Nos tambem garantimos!"
"Deixa que a gente faz melhor!"
```

## SITUACOES ESPECIAIS

### Cliente em Desespero

- Acolher com empatia
- Nao explorar vulnerabilidade
- Explicar realidade com cuidado
- Oferecer suporte humano

### Caso sem Chances

- Ser honesto sobre probabilidades
- Sugerir alternativas se houver
- Nao aceitar caso impossivel
- Documentar a negativa

### Reclamacao de Cliente

- Ouvir integralmente
- Nao ficar defensivo
- Resolver se possivel
- Aprender com o feedback

---

## METRICAS DE ETICA

### O Que Monitorar

- Taxa de reclamacoes
- Satisfacao do cliente
- Expectativas vs realidade
- Devolucoes por insatisfacao

### Alertas Vermelhos

- Cliente surpreso com prazo
- Cliente diz "voces prometeram"
- Reclamacao em redes sociais
- Comentario sobre pressao

### Acoes Corretivas

1. Identificar origem do problema
2. Corrigir comunicacao
3. Treinar equipe
4. Prevenir recorrencia

- Onboarding com etica
- Revisao trimestral
- Casos de estudo

- Rejeitar casos duvidosos

- Nao premiar so por conversao
- Valorizar satisfacao do cliente
