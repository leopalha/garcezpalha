## 1. FILOSOFIA DE PRECIFICAÇÃO

### 1.1 Princípios

```
1. PREÇO FIXO: Cliente sabe quanto vai pagar antes de contratar
2. VALOR PERCEBIDO: Preço reflete o valor entregue, não o custo
3. ESCADA DE VALOR: Opções para diferentes necessidades/bolsos
4. TRANSPARÊNCIA: Sem surpresas, sem custos ocultos
5. URGÊNCIA VALORIZADA: Casos urgentes podem ter acréscimo
```

### 1.2 Estrutura Padrão

```
┌─────────────────────────────────────────────────────────────────────┐
│                    ESCADA DE VALOR PADRÃO                           │
└─────────────────────────────────────────────────────────────────────┘

    PREMIUM ─────────────────────────────────────────── R$ XXX
    │   Tudo do Completo + Extras + Acompanhamento
    │
    COMPLETO ⭐ ──────────────────────────────────────── R$ XX (PADRÃO)
    │   Resolução completa do problema
    │
    ESSENCIAL ────────────────────────────────────────── R$ X
        Solução básica

* O pacote COMPLETO é sempre o mais vendido (80% dos casos)
```

## 3. REGRAS DE PRECIFICAÇÃO DINÂMICA

### 3.1 Fatores de Ajuste

```
┌─────────────────────────────────────────────────────────────────────┐
│                   FATORES DE AUMENTO                                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  FATOR              │ AJUSTE    │ QUANDO APLICAR                    │
│  ──────────────────────────────────────────────────────────────────  │
│  Urgência extrema   │ +30%      │ Cliente precisa em 24h            │
│  Complexidade alta  │ +20-50%   │ Múltiplos réus, situação confusa  │
│  Valor da causa alto│ +20%      │ Causa > R$ 100.000                │
│  Múltiplos processos│ +10% cada │ Cada processo adicional           │
│  Trabalho extra     │ +15%      │ Audiência, perícia, etc.          │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│                   FATORES DE DESCONTO                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  FATOR              │ AJUSTE    │ QUANDO APLICAR                    │
│  ──────────────────────────────────────────────────────────────────  │
│  Pagamento à vista  │ -5%       │ 100% no ato                       │
│  Cliente recorrente │ -10%      │ Já contratou antes                │
│  Indicação          │ -5%       │ Veio por indicação de cliente     │
│  Pacote família     │ -10%      │ Mais de 1 caso da mesma família   │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### 3.2 Fórmula de Cálculo

```
PREÇO FINAL = PREÇO BASE × (1 + FATORES_AUMENTO) × (1 - FATORES_DESCONTO)

EXEMPLO:
- Produto: Desbloqueio Completo (R$ 2.500)
- Urgência extrema: +30%
- Pagamento à vista: -5%

CÁLCULO:
R$ 2.500 × 1,30 × 0,95 = R$ 3.087,50 → Arredondar: R$ 3.100
```

### 3.3 Regras de Arredondamento

```
SEMPRE ARREDONDAR PARA:
├── Múltiplos de R$ 50 (até R$ 1.000)
├── Múltiplos de R$ 100 (R$ 1.000 a R$ 5.000)
└── Múltiplos de R$ 500 (acima de R$ 5.000)

EXEMPLOS:
├── R$ 1.287 → R$ 1.300
├── R$ 2.543 → R$ 2.500
└── R$ 7.234 → R$ 7.500
```

## 4. FORMAS DE PAGAMENTO

### 4.1 Estrutura Padrão

```
PADRÃO (RECOMENDADO):
├── 50% na contratação (entrada)
└── 50% em até 3x (antes do protocolo)

ALTERNATIVAS:
├── À VISTA: 5% de desconto
├── 6X: Sem acréscimo
├── 12X: +10% (juros do gateway)

CONTINGÊNCIA (casos selecionados):
├── Entrada mínima: 20%
├── Êxito: 20-30% do benefício obtido
├── Apenas para: BPC, Revisão INSS, Danos morais altos
```

### 4.2 Exemplos por Produto

```
DESBLOQUEIO R$ 2.500:
├── À vista: R$ 2.375
├── 50% + 3x: R$ 1.250 + 3x R$ 416,67
└── 6x: 6x R$ 416,67

USUCAPIÃO R$ 7.000:
├── À vista: R$ 6.650
├── 50% + 3x: R$ 3.500 + 3x R$ 1.166,67
└── 50% + 6x: R$ 3.500 + 6x R$ 583,33

BPC LOAS R$ 2.000 (contingência):
├── Entrada: R$ 400 (20%)

```
GARANTIA 72H:

"Se em até 72 horas após a contratação você não estiver

### 8.2 Função de Cálculo

```javascript
function calculatePrice(product, package, adjustments = {}) {
  let basePrice = PRICING[product][package];
  
  // Aplicar aumentos
  if (adjustments.urgency) basePrice *= 1.30;
  if (adjustments.complexity) basePrice *= 1.20;
  if (adjustments.highValue) basePrice *= 1.20;
  
  // Aplicar descontos
  if (adjustments.cashPayment) basePrice *= 0.95;
  if (adjustments.returning) basePrice *= 0.90;
  if (adjustments.referral) basePrice *= 0.95;
  
  // Arredondar
  return roundPrice(basePrice);
}

function roundPrice(price) {
  if (price < 1000) return Math.round(price / 50) * 50;
  if (price < 5000) return Math.round(price / 100) * 100;
  return Math.round(price / 500) * 500;
}
```
