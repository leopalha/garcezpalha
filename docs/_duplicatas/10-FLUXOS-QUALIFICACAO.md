## 1. VISÃO GERAL

### 1.1 Objetivo da Qualificação

```
QUALIFICAR = Determinar se podemos ajudar o cliente

Uma boa qualificação:
✓ Evita perda de tempo com casos inviáveis
✓ Permite precificar corretamente
✓ Coleta dados necessários para produção
✓ Gera expectativas realistas
```

### 1.2 Estrutura Padrão

```
┌─────────────────────────────────────────────────────────────────────┐
│                    FLUXO DE QUALIFICAÇÃO                            │
└─────────────────────────────────────────────────────────────────────┘

[1] PERGUNTAS ELIMINATÓRIAS
    └── Se NÃO passa → Dispensar educadamente

[2] PERGUNTAS DE CONTEXTO
    └── Entender detalhes do caso

[3] PERGUNTAS DE COLETA
    └── Dados pessoais e documentação

[4] CLASSIFICAÇÃO DE COMPLEXIDADE
    └── Simples / Padrão / Complexo

[5] RESULTADO
    ├── QUALIFICADO → Ir para proposta
    └── NÃO QUALIFICADO → Explicar e dispensar
```

## 2. PROTEÇÃO FINANCEIRA

### 2.1 Desbloqueio de Conta

```
┌─────────────────────────────────────────────────────────────────────┐
│           QUALIFICAÇÃO: DESBLOQUEIO DE CONTA                        │
└─────────────────────────────────────────────────────────────────────┘

PERGUNTAS ELIMINATÓRIAS:

P1: "Esse dinheiro que foi bloqueado é do seu salário, 
     aposentadoria, pensão ou poupança?"

    ├── SIM → Continua ✓
    └── NÃO (investimento, empresa, MEI, etc)
        → "Infelizmente, esse tipo de valor não tem a mesma 
           proteção legal. Posso te explicar melhor se quiser."
        → DISPENSA ou escalonar

P2: "O bloqueio foi por ordem judicial (processo na justiça)
     ou foi o próprio banco que bloqueou?"

    ├── JUDICIAL → Continua ✓
    ├── BANCO (fraude, irregularidade)
        → "Quando o banco bloqueia por suspeita de fraude, 
           o processo é diferente. Deixa eu entender melhor..."
        → Pode ser outro produto ou escalonar
    └── NÃO SABE
        → "Você recebeu algum documento ou notificação 
           explicando o motivo?"

P3: "Você sabe qual é o motivo da dívida que gerou o bloqueio?
     É pensão alimentícia ou outro tipo de dívida?"

    ├── OUTRA DÍVIDA → Continua ✓
    └── PENSÃO ALIMENTÍCIA
        → "No caso de pensão alimentícia, a lei permite
           o bloqueio do salário. Infelizmente não conseguimos
           ajudar nesse caso específico."
        → DISPENSA

PERGUNTAS DE CONTEXTO:

P4: "Quanto foi bloqueado, aproximadamente?"
    → Registrar valor
    → Se > 100% do salário, pode indicar múltiplas penhoras

P5: "Em qual banco está a conta?"
    → Registrar banco

P6: "Você sabe o número do processo ou nome de quem está
     te cobrando?"
    → Se souber, registrar

P7: "Quando aconteceu o bloqueio?"
    → Registrar data
    → Se há muito tempo, pode ter outras penhoras

DADOS A COLETAR:
├── Nome completo
├── CPF
├── Banco
├── Valor bloqueado
├── Origem do dinheiro (salário/aposentadoria/pensão/poupança)
├── Data do bloqueio
├── Número do processo (se souber)
└── Credor (se souber)

CLASSIFICAÇÃO:
├── SIMPLES: Bloqueio único, valor < salário, tem docs
├── PADRÃO: Bloqueio de valor médio, pode ter múltiplas penhoras
└── COMPLEXO: Múltiplos processos, valor alto, situação confusa

RESULTADO:
├── QUALIFICADO → Propor pacote (Essencial R$ 1.500 / Completo R$ 2.500)
└── NÃO QUALIFICADO → Explicar motivo e dispensar
```

### 2.2 Golpe do PIX

```
┌─────────────────────────────────────────────────────────────────────┐
│              QUALIFICAÇÃO: GOLPE DO PIX                             │
└─────────────────────────────────────────────────────────────────────┘

PERGUNTAS ELIMINATÓRIAS:

P1: "Me conta o que aconteceu. Você foi vítima de um golpe
     ou fraude? Alguém te enganou?"

    ├── SIM (golpe claro) → Continua ✓
    ├── TRANSAÇÃO COMERCIAL (comprou e não recebeu)
        → "Isso é mais uma questão de consumidor. 
           Posso te ajudar de outra forma..."
        → Pode ser outro produto
    └── ERRO DE TRANSFERÊNCIA (mandou pra pessoa errada)
        → "Quando é erro, o caminho é diferente.
           Deixa eu te explicar..."
        → Escalonar ou outro tratamento

P2: "Quando isso aconteceu? Há quantos dias?"

    ├── ATÉ 7 DIAS → Ótimo, MED ainda pode funcionar ✓
    ├── 7-30 DIAS → Ainda dá, mas mais difícil ✓
    ├── 30-90 DIAS → Difícil MED, mas ação judicial sim ✓
    └── MAIS DE 90 DIAS
        → "Quanto mais tempo passa, mais difícil recuperar.
           Mas ainda podemos tentar via judicial."
        → Continua com ressalvas

P3: "Você tem prints das conversas com o golpista?
     Comprovante da transferência?"

    ├── SIM → Ótimo ✓
    └── NÃO
        → "É importante ter provas. Você consegue recuperar
           o comprovante no app do banco?"
        → Se não tiver NADA, fica muito difícil

P4: "Já registrou Boletim de Ocorrência?"

    ├── SIM → Ótimo ✓
    └── NÃO
        → "Vamos precisar fazer o B.O. Podemos te ajudar com isso."

PERGUNTAS DE CONTEXTO:

P5: "Qual foi o valor que você perdeu?"
    → Registrar valor

P6: "Para qual banco foi o dinheiro?"
    → Registrar banco destino

P7: "De qual banco você transferiu?"
    → Registrar banco origem

P8: "Você tem o nome ou CPF de quem recebeu?"
    → Registrar se tiver

DADOS A COLETAR:
├── Nome completo
├── CPF
├── Banco origem
├── Banco destino
├── Valor perdido
├── Data do golpe
├── Tipo de golpe (falso boleto, falso funcionário, etc)
├── Tem prints? (sim/não)
├── Tem B.O.? (sim/não)
└── Dados do golpista (se tiver)

CLASSIFICAÇÃO:
├── SIMPLES: Até 7 dias, tem provas, valor médio
├── PADRÃO: 7-30 dias, tem provas
└── COMPLEXO: +30 dias, poucas provas, valor alto

RESULTADO:
├── QUALIFICADO → Propor pacote (MED R$ 1.500 / Completo R$ 2.500 / Premium R$ 4.000)
└── NÃO QUALIFICADO → Explicar dificuldades, sugerir B.O.
```

### 2.3 Negativação Indevida

```
┌─────────────────────────────────────────────────────────────────────┐
│           QUALIFICAÇÃO: NEGATIVAÇÃO INDEVIDA                        │
└─────────────────────────────────────────────────────────────────────┘

PERGUNTAS ELIMINATÓRIAS:

P1: "Você reconhece essa dívida? É algo que você realmente deve?"

    ├── NÃO RECONHECE → Continua ✓
    ├── JÁ PAGOU → Continua ✓
    ├── RECONHECE E DEVE
        → "Se a dívida existe, a negativação não é indevida.
           Posso te ajudar a negociar, se quiser."
        → Outro produto ou dispensa

P2: "Você sabe há quanto tempo existe essa dívida?"

    ├── MENOS DE 5 ANOS → Continua ✓
    └── MAIS DE 5 ANOS
        → "Dívidas com mais de 5 anos estão prescritas!
           Você pode ter direito a indenização."
        → Continua (caso ainda melhor)

P3: "Você tem algum comprovante de que não deve ou já pagou?"

    ├── SIM → Ótimo ✓
    └── NÃO
        → "Vamos trabalhar com o que temos. Podemos 
           solicitar comprovação da empresa."
        → Continua

PERGUNTAS DE CONTEXTO:

P4: "Qual empresa te negativou?"
    → Registrar empresa

P5: "Qual o valor da suposta dívida?"
    → Registrar valor

P6: "É só uma negativação ou tem mais de uma?"
    → Se múltiplas, pode aumentar valor e indenização

DADOS A COLETAR:
├── Nome completo
├── CPF
├── Empresa(s) que negativou
├── Valor(es) da dívida
├── Data da negativação
├── Motivo alegado
├── Tem comprovante de pagamento? (se aplicável)
└── Quantas negativações no total

CLASSIFICAÇÃO:
├── SIMPLES: Uma negativação, valor baixo
├── PADRÃO: Uma ou duas negativações
└── COMPLEXO: Múltiplas negativações, valores altos

RESULTADO:
├── QUALIFICADO → Propor pacote (R$ 1.200 a R$ 2.500)
└── NÃO QUALIFICADO → Explicar e sugerir negociação
```

## 3. PROTEÇÃO PATRIMONIAL

### 3.1 Usucapião

```
┌─────────────────────────────────────────────────────────────────────┐
│               QUALIFICAÇÃO: USUCAPIÃO                               │
└─────────────────────────────────────────────────────────────────────┘

PERGUNTAS ELIMINATÓRIAS:

P1: "Há quanto tempo você mora/usa esse imóvel?"

    ├── 5-10 ANOS → Possível usucapião ordinário ✓
    ├── 10-15 ANOS → Usucapião ordinário/extraordinário ✓
    ├── 15+ ANOS → Usucapião extraordinário ✓
    └── MENOS DE 5 ANOS
        → "Para usucapião, geralmente precisa de pelo menos
           5 anos de posse. Tem outra documentação do imóvel?"
        → Pode não qualificar ou outro produto

P2: "O imóvel é de quem legalmente? Você sabe?"

    ├── PARTICULAR (pessoa física/jurídica) → Continua ✓
    ├── HERANÇA NÃO REGULARIZADA → Pode ser usucapião ou inventário
    └── PÚBLICO (prefeitura, estado, união)
        → "Imóveis públicos não podem ser usucapidos.
           Mas existem outros caminhos de regularização."
        → Escalonar ou outro produto

P3: "Existe alguém disputando esse imóvel com você?"

    ├── NÃO → Ótimo, extrajudicial possível ✓
    └── SIM
        → "Se há disputa, o usucapião precisa ser judicial.
           É mais demorado, mas ainda possível."
        → Continua com ressalvas

P4: "O imóvel tem matrícula no cartório?"

    ├── SIM → Retificação pode ser suficiente
    ├── NÃO → Usucapião necessário
    └── NÃO SABE
        → "Vamos verificar isso. Você sabe o endereço completo?"

PERGUNTAS DE CONTEXTO:

P5: "Qual o tamanho aproximado do imóvel/terreno?"
    → Registrar área

P6: "É casa, apartamento, terreno ou fazenda?"
    → Registrar tipo

P7: "Você paga IPTU? Tem comprovantes?"
    → Provas de posse

P8: "Tem contas de luz/água antigas no seu nome?"
    → Provas de posse

P9: "Em qual cidade fica o imóvel?"
    → Registrar localização

DADOS A COLETAR:
├── Nome completo
├── CPF
├── Endereço do imóvel
├── Tempo de posse
├── Tipo de imóvel (casa, terreno, etc)
├── Área aproximada
├── Tem IPTU? (anos)
├── Tem contas no nome? (anos)
├── Existe disputa?
├── Imóvel tem matrícula?
└── Estado/Cidade

CLASSIFICAÇÃO:
├── SIMPLES: +15 anos, sem disputa, tem provas, extrajudicial
├── PADRÃO: 10-15 anos, sem disputa, algumas provas
└── COMPLEXO: Disputa, provas fracas, judicial necessário

RESULTADO:
├── QUALIFICADO → Propor pacote (R$ 5.000 a R$ 10.000)
└── NÃO QUALIFICADO → Explicar e sugerir alternativas
```

### 3.2 Holding Familiar

```
┌─────────────────────────────────────────────────────────────────────┐
│             QUALIFICAÇÃO: HOLDING FAMILIAR                          │
└─────────────────────────────────────────────────────────────────────┘

PERGUNTAS ELIMINATÓRIAS:

P1: "Você tem imóveis, investimentos ou empresas que quer proteger?"

    ├── SIM → Continua ✓
    └── NÃO
        → "A holding é para quem tem patrimônio.
           Qual seria seu objetivo exatamente?"
        → Entender melhor ou dispensa

P2: "Qual o valor aproximado do patrimônio que quer proteger?"

    ├── ACIMA DE R$ 500K → Holding faz sentido ✓
    └── ABAIXO DE R$ 500K
        → "Para patrimônios menores, a holding pode não
           compensar pelos custos. Talvez testamento ou
           doação seja mais indicado."
        → Pode ser outro produto

PERGUNTAS DE CONTEXTO:

P3: "Qual seu objetivo principal? Proteger de credores,
     planejar herança, economia de impostos?"
    → Registrar objetivo

P4: "Quantos imóveis você tem?"
    → Registrar quantidade

P5: "Você tem empresa própria?"
    → Registrar se tem

P6: "Quantos herdeiros você tem?"
    → Registrar quantidade

P7: "Você é casado? Qual regime de bens?"
    → Registrar estado civil e regime

DADOS A COLETAR:
├── Nome completo
├── CPF
├── Estado civil
├── Regime de bens
├── Número de herdeiros
├── Patrimônio aproximado
├── Quantidade de imóveis
├── Tem empresa?
├── Objetivo principal
└── Estado

CLASSIFICAÇÃO:
├── SIMPLES: Até 3 imóveis, família pequena
├── PADRÃO: 3-10 imóveis ou + empresa
└── COMPLEXO: Muitos imóveis, empresas, família grande

RESULTADO:
├── QUALIFICADO → Propor consultoria + holding (R$ 8.000 a R$ 15.000)
└── NÃO QUALIFICADO → Sugerir alternativas (testamento, doação)
```

## 4. PROTEÇÃO SAÚDE

### 4.1 Plano de Saúde

```
┌─────────────────────────────────────────────────────────────────────┐
│            QUALIFICAÇÃO: PLANO DE SAÚDE                             │
└─────────────────────────────────────────────────────────────────────┘

PERGUNTAS ELIMINATÓRIAS:

P1: "O plano negou formalmente? Você tem a negativa por escrito?"

    ├── SIM → Continua ✓
    ├── NÃO (só demora)
        → "Se ainda não negou formalmente, podemos te orientar
           a solicitar por escrito. Quer que eu explique?"
        → Continua com orientação
    └── NÃO SABE
        → "Você pediu algo e eles disseram não?"

P2: "O procedimento foi pedido por um médico? Tem o pedido médico?"

    ├── SIM → Continua ✓
    └── NÃO
        → "Sem pedido médico fica difícil. O plano só é
           obrigado a cobrir o que médico prescreve."
        → Pode não qualificar

P3: "Seu plano está ativo? Está em dia com as mensalidades?"

    ├── SIM → Continua ✓
    └── NÃO (cancelado ou inadimplente)
        → "Se o plano foi cancelado por falta de pagamento,
           a situação muda. Me conta mais..."
        → Avaliar caso a caso

PERGUNTAS DE CONTEXTO:

P4: "O que o plano negou? Cirurgia, exame, medicamento, terapia?"
    → Registrar tipo

P5: "Qual o motivo da negativa alegado pelo plano?"
    → Registrar motivo

P6: "Qual o nome do seu plano de saúde?"
    → Registrar operadora

P7: "É urgente? Tem data marcada para o procedimento?"
    → Define urgência da liminar

DADOS A COLETAR:
├── Nome completo
├── CPF
├── Operadora do plano
├── Tipo de plano (individual, empresarial)
├── O que foi negado
├── Motivo da negativa
├── Tem pedido médico?
├── Tem negativa por escrito?
├── É urgente?
└── Data prevista do procedimento

CLASSIFICAÇÃO:
├── SIMPLES: Procedimento no rol ANS, negativa clara
├── PADRÃO: Procedimento comum, precisa argumentar
└── COMPLEXO: Fora do rol, experimental, alto custo

RESULTADO:
├── QUALIFICADO → Propor pacote (R$ 2.500 a R$ 5.000)
└── NÃO QUALIFICADO → Orientar outros caminhos (ANS, Procon)
```

### 4.2 BPC LOAS

```
┌─────────────────────────────────────────────────────────────────────┐
│               QUALIFICAÇÃO: BPC LOAS                                │
└─────────────────────────────────────────────────────────────────────┘

PERGUNTAS ELIMINATÓRIAS:

P1: "Você é idoso (65+) ou pessoa com deficiência?"

    ├── IDOSO 65+ → Continua ✓
    ├── DEFICIENTE → Continua ✓
    └── NENHUM
        → "O BPC LOAS é só para idosos acima de 65 anos
           ou pessoas com deficiência. Tem outra situação?"
        → Dispensa ou outro produto

P2: "Você ou sua família têm renda? Quanto aproximadamente
     por pessoa da casa?"

    ├── ATÉ 1/4 SM POR PESSOA → Qualifica ✓
    ├── ATÉ 1/2 SM POR PESSOA → Pode qualificar (judicial) ✓
    └── ACIMA
        → "A renda familiar pode estar acima do limite.
           Mas existem exceções. Quantas pessoas moram
           na sua casa?"
        → Avaliar composição familiar

P3: "Você já pediu o BPC no INSS?"

    ├── SIM, NEGADO → Perfeito para ação judicial ✓
    ├── SIM, AGUARDANDO → Orientar aguardar ou entrar judicial
    └── NÃO
        → "Geralmente precisa pedir primeiro no INSS.
           Quer que eu te oriente como fazer?"
        → Pode fazer administrativo primeiro

PERGUNTAS DE CONTEXTO:

P4: "Quantas pessoas moram na sua casa?"
    → Calcular renda per capita

P5: "Qual a renda total da família?"
    → Calcular se qualifica

P6: "Você tem o CadÚnico atualizado?"
    → Importante para o processo

P7: "Se for deficiência, você tem laudos médicos?"
    → Necessário para perícia

DADOS A COLETAR:
├── Nome completo
├── CPF
├── Idade
├── Tipo (idoso ou deficiente)
├── Se deficiente: qual deficiência
├── Composição familiar
├── Renda familiar total
├── Tem CadÚnico?
├── Já pediu no INSS?
├── Tem carta de indeferimento?
└── Tem laudos médicos? (se deficiente)

CLASSIFICAÇÃO:
├── SIMPLES: Critérios claros, já negado, docs completos
├── PADRÃO: Renda no limite, precisa argumentar
└── COMPLEXO: Renda acima, deficiência questionável

RESULTADO:
├── QUALIFICADO → Propor pacote (R$ 2.000 a R$ 2.500)
└── NÃO QUALIFICADO → Orientar CadÚnico, pedido no INSS
```

## 5. RESUMO: PERGUNTAS ESSENCIAIS

### 5.1 Mínimo para Qualificar (por área)

| Área | Perguntas Mínimas |
|------|-------------------|
| Desbloqueio | Origem do dinheiro + Tipo de bloqueio + Não é pensão |
| Golpe PIX | Foi golpe + Quando foi + Tem provas |
| Negativação | Não reconhece/já pagou + Qual empresa |
| Usucapião | Tempo de posse + Não é público + Sem disputa |
| Plano Saúde | Tem negativa + Tem pedido médico + Plano ativo |
| BPC LOAS | Idoso/deficiente + Renda baixa + Já pediu INSS |

### 5.2 Dados Sempre Coletados

```
PARA TODOS OS CASOS:
├── Nome completo
├── CPF
├── Telefone (já tem pelo WhatsApp)
├── Estado onde mora
└── Email (opcional, mas útil)
```

## 6. SCRIPTS DE DISPENSA

### 6.1 Caso Não Qualifica

```
GENÉRICO:
"[Nome], pelo que você me contou, infelizmente esse caso 
específico não se encaixa nos serviços que oferecemos.

[Explicar brevemente o motivo]

Recomendo que você procure [alternativa].

Se surgir alguma situação nas áreas que atendemos 
(contas bloqueadas, golpes, imóveis, plano de saúde), 
é só me chamar. Boa sorte!"
```

```
PENSÃO ALIMENTÍCIA:
"Entendo sua situação, mas quando o bloqueio é por pensão 
alimentícia, a lei permite que o salário seja penhorado.
Infelizmente não conseguimos ajudar nesse caso.

Se você acha que o valor da pensão está errado, o caminho 
é uma ação de revisão de alimentos. Posso indicar alguém 
se quiser."
```

```
RENDA ALTA (BPC):
"Pelo cálculo que fizemos, a renda da sua família está 
acima do limite para o BPC. Mas existem outras formas de 
aposentadoria ou benefício que você pode ter direito.

Recomendo procurar o CRAS da sua cidade ou um advogado 
previdenciário."
```
