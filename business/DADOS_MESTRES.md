## 16. UNIT ECONOMICS

### 16.1 Exemplo: Consultoria Imobiliária Completa

```
Receita: R$ 2.500
(-) Comissão parceiro (20%): -R$ 500
(-) Gateway pagamento (2%): -R$ 50
(-) Custo IA (~R$ 5/consulta): -R$ 10
= Margem Bruta: R$ 1.940 (77,6%)

(-) Custo fixo rateado: -R$ 50
= Margem Líquida: R$ 1.890 (75,6%)
```

### 16.2 LTV:CAC Ratio

```
LTV = R$ 12.000 (lifetime value médio)
CAC = R$ 450 (custo de aquisição médio)

LTV:CAC = 12.000 / 450 = 26,7:1

Benchmark: > 3:1 (excelente)
Resultado: 26,7:1 (excepcional)
```
